Fake Invite Detection

If a member leaves within this many minutes of joining, the invite is counted as fake instead of left. Fake invites reduce net invites just like leaves do, but are tracked separately so you can spot abuse patterns.

minutes (default: 10)
Link Monitor (Auto-Delete)

Automatically deletes any message containing a Discord server invite link (discord.gg/...). Server admins and staff are exempt. The user gets a 5-second warning message that disappears on its own. Normal URLs and other links are not affected.

Paste one or more Channel IDs separated by commas, or type all to monitor every channel in the server. Right-click a channel in Discord → Copy Channel ID (requires Developer Mode).
